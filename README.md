# TeamDock — Collaborative Task Management

TeamDock is a high-performance, premium project management and team collaboration dashboard. Built with Next.js and Tailwind CSS, it offers a glassmorphic aesthetic and a seamless user experience for managing tasks, projects, and team members.

## 🚀 Features

- **Dynamic Dashboard**: Monitor total projects, tasks, and overdue items at a glance.
- **Project Management**: Create, view, and manage projects with role-based controls.
- **Task Tracking**: Stay on top of your work with assigned task lists and due date tracking.
- **Team Collaboration**: View and manage team members and their roles.
- **Private Notepad**: A secure, auto-saving notepad for your quick thoughts.
- **Modern UI**: Dark/Light mode support with a premium glassmorphic design.

## 🛠 Tech Stack

- **Framework**: [Next.js 14](https://nextjs.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **State Management**: Custom Store Provider
- **Deployment**: Optimized for [Vercel](https://vercel.com/)

## 🏁 Getting Started

### Prerequisites

- Node.js (LTS recommended)
- pnpm

### Installation

1. Clone the repository:
   ```bash
   git clone <your-repo-url>
   ```

2. Install dependencies:
   ```bash
   pnpm install
   ```

3. Start the development server:
   ```bash
   pnpm dev
   ```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## 🚢 Deployment

The easiest way to deploy TeamDock is to use the [Vercel Platform](https://vercel.com/new).

Check out the [Next.js deployment documentation](https://nextjs.org/docs/deployment) for more details.
